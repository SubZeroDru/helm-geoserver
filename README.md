This chart includes the following features:

  - Ingress
  - ECW
  - Mrsid
  - Non root operation
  - NFS based storageClass volume claim
  - Included test pod yaml

### Installation
$ helm install MY_RELEASE . 

### Uninstall
$ helm del MY_RELEASE
